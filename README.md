Git-Butler
====

[Documentation can be found here](./docs/content.md)

## Install

```sh
curl -fsSL https://github.com/ylallemant/git-butler/raw/0.0.2/install.sh | bash
```

## Tests

### Run

```bash
go test -cover ./...
```

### Coverage

Enable visual coverage feedback in `vscode` :

```json
{
    "go.coverOnSave": true
}
```
